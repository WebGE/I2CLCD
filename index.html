PCF2119 : LCD controllers/drivers

Le PCF2119 est un circuit intégré destiné à la commande des matrices de points 5x8 des lcd 2x16 ou 1x32 via un bus I2C.

  • Exemple : Afficheur MIDAS - MC21605E6W-SPTLYI - LCD 2X16 STN LED B/L Y/VERT I2C I/F
  • Documentation: pdf

    • Il peut être utiliser comme dans l'exemple ci-dessous.

Classe I2CLCD

Contenu du dossier à télécharger

Le dossier à télécharger (au format zip ou tar.gz) contient le répertoire d'un projet Microsoft Visual Studio Professionnel composé :

  • Du code de la classe I2CLCD.
  • D'un exemple d'utilisation de cette classe.

Les fichiers program.cs et DriversI2CLCD.cs pourront être recompilés avec Microsoft Visual Studio Express.

Utilisation

  1. Créer un objet en utilisant un des constructeurs I2CLcd.
  2. Initialiser le LCD avec la méthode Init.
  3. Effacer l'écran avec la méthode ClearScreen.
  4. Utiliser les autres méthodes pour gérer l'affichage.

Constructeurs

Syntaxe Description
I2CLcd() Instancie un objet "I2CLcd" avec les paramètres :
Fabricant = Midas
Fréquence du bus: 100kHz
I2CLcd(UInt16 I2C_Add_7bits) Instancie un objet "I2CLcd".
I2C_Add_7bits est l'adresse I2C sur 7 bits attribuée à l'afficheur.
Fréquence du bus: 100kHz.
I2CLcd(LcdManufacturer ManufacturerName) Instancie un objet "I2CLcd". ManufacturerName est le nom du fabricant.
Il correspond à l'adresse I2C du LCD :
  • Midas (0x3a)
  • Batron (0x3b)
Fréquence du bus: 100kHz.
I2CLcd(UInt16 I2C_Add_7bits, UInt16 Frequency) Instancie un objet "I2CLcd".
I2C_Add_7bits est l'adresse I2C sur 7 bits attribuée à l'afficheur.
Frequency est à définir dans l'intervalle [100kHz,400kHz].
I2CLcd(LcdManufacturer ManufacturerName, UInt16 Frequency) Instancie un objet "I2CLcd". ManufacturerName est le nom du fabricant.
Il correspond à l'adresse I2C du LCD :
  • Midas (0x3a)
  • Batron (0x3b)
Frequency est à définir dans l'intervalle [100kHz,400kHz].

Enumérations

Syntaxe Description
LcdManufacturer MIDAS : 0x3a (par défaut)
BATRON : 0x3B
CursorType Hide : caché (par défaut)
Underline : soulignement
Blink : clignotant

Méthodes publiques

Syntaxe Description
void ClearScreen() Efface l'écran (2 x 16 caractères).
void Init() Initialise l'afficheur en mode 2 x 16 caractères. A utiliser en premier après la construction de l'objet.
void LineBegin(byte x)

Fixe le début de la ligne.

x: position en x sur l'afficheur (0..15).
void PutChar(byte x_pos, byte y_pos, byte z_char)

Ecrit un caractère (z_char) aux coordonnées (x_pos, y_pos) de l'afficheur.

x_pos: position en x sur l'afficheur (0..15).

y_pos: position en y sur l'afficheur (0..1).

z_char: caractère à afficher.
void PutString(byte x_pos, byte y_pos, string Text)

Ecrit une chaîne de caractères (Text) aux coordonnées (x_pos, y_pos) de l'afficheur.

x_pos: position en x sur l'afficheur (0..15).

y_pos: position en y sur l'afficheur (0..1).

Text: Chaîne à afficher.
void SelectCursor(CursorType posCursorx)

Sélection du curseur

CursorType: Hide, Underline, Blink
void SetBacklight(byte bright)

Réglage du contraste de l'aficheur

bright: 20 dark -> 0 light
