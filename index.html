PCF219 : LCD controllers/drivers

Le PCF2119 est un circuit intégré destiné à la commande des matrices de points 5x8 des lcd 2x16 ou 1x32 via un bus I2C.

  • Exemple : Afficheur MIDAS - MC21605E6W-SPTLYI - LCD 2X16 STN LED B/L Y/VERT I2C I/F

    • Il peut être utiliser comme dans l'exemple ci-dessous.

Exemple d'IHM

Les huit Leds et les huit boutons-poussoir sont commandés par deux PCF8574.

Secret Labs
  • Microcontrôleur : STM32F405RG 32 bits à architecture Cortex-M4 cadencé à 168 MHz.
  • RAM : 100 KB.
  • Flash : 384 KB.
  • Port Ethernet 10Mbps.
  • E/S numériques : 14
  • Entrées analogiques : 6
  • IDE : Microsoft Visual Studio (professionnel ou express)
  • Framework : micro framework .NET (NETMF)
  • Langage de programmation : C#

Classe PCF8574

Contenu du dossier à télécharger

Le dossier à télécharger (au format zip ou tar.gz) contient le répertoire d'un projet Microsoft Visual Studio Professionnel composé :

  • Du code de la classe PCF8574.
  • D'un exemple d'utilisation de cette classe.

Les fichiers program.cs et DriversPCF8574.cs pourront être recompilés avec Microsoft Visual Studio Express.

Utilisation

  1. Créer un objet en utilisant le constructeur PCF8574.
  2. Utiliser la méthode WriteByte pour écrire sur le port relié aux Leds et la méthode ReadByte pour lire le port relié aux boutons-poussoir.
